最怕误入歧途而不自知，省点子弹等待猎物

这两天一直在折腾，将 Hexo 部署到 Github Pages 上就已经花费了大半的心力。即使稍微自学过一点儿 WordPress 网站的搭建，其实过程当中还是云里雾里，大半还是借助着网络资讯的发达，最后才勉强实现第一个里程碑：拥有一个个人博客。

接着就是选博客主题，安装调试，期间大小问题不断，只得依靠薄弱的网络知识以及强大的搜索引擎去解决层出不穷的拦路虎，我实在是担心，不知道哪里冒出来的问题又会将我的两分钟热度浇灭。而现在我又关心起 Github 在国内访问速度不稳定，偶然间看到 Coding 上也能免费部署静态网站，就跃跃欲试地开始撸起袖子搜教程，最后没有如预期般的顺利。

这一篇则是写于想部署网站在 Coding 上却求而不得之后，因为察觉到我在当前一个时间与精力都不允许的境况下，追求一个本不在考虑范围内的东西。换句话说就是，我其实偏离了自己当初的设想。

如果拍着胸脯认真地问自己一句：目前最紧迫、最必须完成的是上述说的那些事情吗？

我可以肯定地说不是。如果真的紧迫、必须搞定的话，不应该在这儿折腾免费的东西，而是直接购买阿里或者腾讯的服务，采用花钱定制的方式获得理想中的效果。免费的东西意味着需要额外的学习成本。

我的本意，不过是想在网络上有一个属于自己的角落，可以持续地输出原创内容，并且获得反馈（短时间内很难实现）。持续输出，获得反馈，然后不断地学习，拓宽思维的广度与深度，这就是我的目的。

对，持续输出才应该是我目前最紧迫、最需要关注的方面，除此之外的其他，在精力不允许的情况之下只能靠后。于是，现在的问题就转化成了：如何让自己做到高质量持续输出？

首先就是，人类是尤其热衷于排名的，不信你看心理学专业都有学校排名。我参加 30 天每天 200 下俯卧撑挑战最直接的原因，就只是想看看和那三个人相比，自己能够排到第几。一段时间后，原先触发这一特定行为的动机有可能会变成挑战自我，或者是继续超越他人，而初衷是为了和他人比较，这一点是确定的。

第二，将目标具体到某个数字是相对有效的，“尽量完成”、“全力以赴”、“坚持到底”……这样的词对于目标的达成是毫无用处的，人类很容易被自认为自己具备的精神所感动到。数字是冰冷的，但是它很理性。

心理学中有一个简单的行为，越是得到强化的事情我们就越倾向于多做，而那些会受到惩罚或被忽视的事情，我们往往就会少做。在很大程度上我们就是按照这样的原则行事的，一个行为得到的正反馈越多越及时，我们就会渴望越来越多、越来越及时的正反馈。

每天 1000 下俯卧撑对我来说是天方夜谭，也许尝试过那么一次之后，“我做不到”、“打死我也办不到”这样的的负能量就直接将人劝退了。

每天 200 下可能算得上一个洪水猛兽，但却是努努力就能将其收服的目标，这样子每天都能得到积极反馈、长期坚持下去心理上又能收获某种自信，想不坚持都难。

摘自：《报告！我也完成了 30 天每天 200 下俯卧撑挑战》

借鉴 30 天每天 200 下俯卧撑挑战时的感悟，目前就应该暂时抛弃不切实际的想法，也就是对高质量的追求，而专注于持续输出上，即如何完成“每天 200 下俯卧撑”并且将其持续“30 天”。

《矛盾论》里说到，“在复杂的事物的发展过程中，有许多的矛盾存在，其中必有一种是主要的矛盾，由于它的存在和发展，规定或影响着其他矛盾的存在和发展”。

做俯卧撑不需要到健身房，一个普通的瑜伽垫就可以。输出内容不需要在多精美的网站上，能实现最基本的功能就够了。运动是为了拥有更加健康、更加强壮的体魄，持续输出是为了不断地学习，强迫大脑进行总结和反思，这样才能更快地成长，让自己有能力处理更多维度的问题。

慢但坚持就是快，少却持续即为多。
